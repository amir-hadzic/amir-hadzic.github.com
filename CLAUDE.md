## Overview

Personal blog at www.ahadzic.com — an Astro static site with Tailwind CSS, deployed to GitHub Pages via GitHub Actions.

DNS is managed through Cloudflare (CNAME pointing to amir-hadzic.github.io).

## Build & Development

```bash
# Local development server
npm run dev

# Build site to dist/
npm run build

# Preview production build
npm run preview
```

## Deployment

Pushes to `master` trigger `.github/workflows/deploy.yml` which builds the site and deploys to GitHub Pages. The `public/CNAME` file ensures the custom domain is preserved on each deploy.

Note: The lockfile is generated on macOS ARM. CI uses `npm install` (not `npm ci`) to resolve platform-specific native dependencies for Linux.

## Architecture

- **Astro 5** static site with Tailwind CSS v4 (via @tailwindcss/vite), MDX support, and @astrojs/sitemap
- **Layouts**: `src/layouts/BaseLayout.astro` (site shell with nav) → `src/layouts/PostLayout.astro` (wraps blog posts)
- **Pages**: `src/pages/` — index, archive, [...slug] (dynamic blog post route), rss.xml
- **Posts**: `src/content/posts/` — markdown files with YAML front matter (title, date, optional draft, optional bookreview). Collection defined in `src/content.config.ts`
- **Drafts**: Set `draft: true` in frontmatter to exclude a post from the site while keeping it in the repo
- **Helpers**: `src/lib/posts.ts` — getSortedPosts() (filters drafts, sorts by date), getPostSlug() for date-based URL generation (YYYY/MM/DD/slug)
- **Styles**: `src/styles/global.css` — Tailwind imports, custom theme (Inter/Lora/JetBrains Mono fonts), `.prose` styles for post content
- **Static assets**: `public/` — images, JS, favicon, CNAME
- **Feeds**: `/rss.xml` (via @astrojs/rss), sitemap auto-generated by @astrojs/sitemap

## Adding a new post

Create a markdown file in `src/content/posts/` with the naming convention `YYYY-MM-DD-slug.md`:

```markdown
---
title: Post title
date: YYYY-MM-DD
draft: true  # optional, omit when ready to publish
---

Content here.
```
