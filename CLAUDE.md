## Overview

Personal blog at www.randomshouting.com — an Astro static site with Tailwind CSS, hosted on GitHub Pages.

## Build & Development

```bash
# Local development server (port 4000)
npm run dev

# Build site to dist/
npm run build

# Preview production build
npm run preview
```

## Architecture

- **Astro 5** static site with Tailwind CSS v4 (via @tailwindcss/vite), MDX support, and @astrojs/sitemap
- **Layouts**: `src/layouts/BaseLayout.astro` (site shell with nav) → `src/layouts/PostLayout.astro` (wraps blog posts)
- **Pages**: `src/pages/` — index, projects, hire-me, archive, canvastyper, [...slug] (dynamic blog post route), rss.xml
- **Posts**: `src/content/posts/` — markdown files with YAML front matter (title, date, optional bookreview). Collection defined in `src/content.config.ts`
- **Helpers**: `src/lib/posts.ts` — getSortedPosts(), getPostSlug() for date-based URL generation (YYYY/MM/DD/slug)
- **Styles**: `src/styles/global.css` — Tailwind imports, custom theme colors, `.prose` styles for post content
- **Static assets**: `public/` — images, JS (CanvasTyper), favicon, CNAME
- **Feeds**: `/rss.xml` (via @astrojs/rss), sitemap auto-generated by @astrojs/sitemap
